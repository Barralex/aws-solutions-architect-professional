# 10.9 AWS Glue

## Qué es

- Servicio **ETL managed** (Extract, Transform, Load)
- Serverless
- Preparar y transformar datos para analytics

## Dos componentes

### 1. Glue ETL
- Extract de S3, RDS, etc.
- Transform dentro de Glue
- Load a destinos (Redshift, S3, etc.)

### 2. Glue Data Catalog
- Catálogo de metadata de tus datasets
- **Crawlers** escanean tus data sources
- Sources: S3, RDS, DynamoDB, JDBC compatible
- Escribe metadata: databases, tables, columns

## Servicios que usan Glue Data Catalog

- Amazon Athena
- Amazon Redshift Spectrum
- Amazon EMR

## Para el examen

| Ves en pregunta | Respuesta |
|-----------------|-----------|
| ETL serverless | AWS Glue |
| Descubrir schema de datos en S3 | Glue Crawler + Data Catalog |
| Transformar datos antes de analytics | Glue ETL |
| Catálogo de metadata centralizado | Glue Data Catalog |
