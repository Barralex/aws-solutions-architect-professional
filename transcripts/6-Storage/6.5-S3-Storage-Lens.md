# 6.5 Amazon S3 - Storage Lens

## Qué es
- Dashboard de análisis de storage a nivel Organization
- Visualiza: usage, cost, data protection, activity
- Scope: Organization, accounts, regions, buckets, prefixes

## Default Dashboard
- Pre-configurado por AWS
- Multi-region, multi-account
- No se puede borrar, solo deshabilitar

## Tipos de Métricas
| Tipo | Qué muestra |
|------|-------------|
| Summary | Storage bytes, object count |
| Cost Optimization | Non-current versions, incomplete multi-part |
| Data Protection | Versioning enabled, MFA delete, encryption, replication |
| Access Management | Bucket ownership settings |
| Event | S3 event notifications config |
| Performance | Transfer Acceleration enabled |
| Activity | Requests, bytes downloaded/uploaded |
| Status Code | 200 OK, 403 Forbidden, etc |

## Free vs Paid
| | Free | Advanced (paid) |
|---|------|-----------------|
