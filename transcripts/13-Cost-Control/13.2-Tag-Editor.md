# 13.2 AWS Tag Editor

## Conceptos Clave

### Qué es
- Herramienta para **buscar recursos y editar tags en bulk**
- Busca en **todas las regiones**
- Puede filtrar recursos **taggeados o sin taggear**

### Qué puede hacer
- **Agregar** tags a múltiples recursos a la vez
- **Actualizar** tags existentes
- **Eliminar** tags

### Tag Editor vs Cost Allocation Tags
| Herramienta | Función |
|---|---|
| **Tag Editor** | Buscar/editar tags en recursos |
| **Cost Allocation Tags** | Ver tags como columnas en billing |
