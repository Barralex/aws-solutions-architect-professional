# 11.4 - AWS X-Ray

## Transcript
AWS X-Ray: distributed tracing for microservices. High-level overview for SA Pro exam.

## Key Concepts

### What is X-Ray
- Visual analysis of applications
- Tracing capability across microservices
- Shows latency, error rates, network call flow
- Find WHERE an issue originated in a distributed system

### Integrations
- **EC2**: install X-Ray agent
- **ECS**: install X-Ray agent or Docker container
- **Lambda**: just tick a box
- **Elastic Beanstalk**: agent auto-installed if enabled
- **API Gateway**: debug errors like 504 timeouts

### IAM
- X-Ray agents/services need correct IAM permissions to talk to X-Ray

### Exam Rule
- "Distributed tracing" or "troubleshooting at request level" → X-Ray
- More focus on Developer exam, SA Pro only needs high-level
