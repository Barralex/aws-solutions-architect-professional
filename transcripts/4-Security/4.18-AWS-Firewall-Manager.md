# 4.18 AWS Firewall Manager

**Curso:** AWS Solutions Architect Professional (SAP-C02)
**Sección:** 4 - Security
**Duración:** ~4 min

---

## Resumen

Firewall Manager gestiona reglas de firewall en TODAS las cuentas de una AWS Organization desde un solo lugar. Aplica políticas de WAF, Shield Advanced, Security Groups y Network Firewall automáticamente, incluso a recursos nuevos.

---

---

## Qué puede gestionar

| Servicio | Recursos protegidos |
|----------|---------------------|
| **WAF rules** | ALB, API Gateway, CloudFront |
| **Shield Advanced** | ALB, CLB, NLB, Elastic IP, CloudFront |
| **Security Groups** | EC2, ALB, ENI en VPC |
| **Network Firewall** | VPC level |
| **Route 53 Resolver** | DNS Firewall |

---

## Feature clave: Auto-aplicación

```
Antes de Firewall Manager:
  - Creas ALB en cuenta 5 → manualmente configuras WAF

Con Firewall Manager:
  - Creas ALB en cuenta 5 → WAF se aplica AUTOMÁTICAMENTE
```

Las políticas se aplican a recursos nuevos sin intervención manual.

---

## WAF vs Shield vs Firewall Manager

| Servicio | Cuándo usarlo |
|----------|---------------|
| **WAF** | Protección única, una cuenta, reglas específicas |
| **Shield Advanced** | Protección DDoS avanzada + DRT + cost protection |
| **Firewall Manager** | Gestión centralizada en MÚLTIPLES CUENTAS |

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   WAF          →  Define las reglas                    │
│   Shield       →  Protección DDoS                      │
│   Firewall Mgr →  Aplica todo a TODAS las cuentas      │
│                                                         │
│   Los 3 se usan JUNTOS para protección completa        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

## Puntos Clave para el Examen

1. **Firewall Manager** = gestión centralizada multi-cuenta (Organizations)
2. **Políticas regionales** aplicadas a todas las cuentas
3. **Auto-aplica** reglas a recursos nuevos automáticamente
4. Gestiona: WAF, Shield Advanced, Security Groups, Network Firewall, DNS Firewall
5. **"Manage security rules across all accounts"** → Firewall Manager
