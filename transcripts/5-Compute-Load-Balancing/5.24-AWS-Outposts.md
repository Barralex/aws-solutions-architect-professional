# 5.24 AWS Outposts

## TL;DR
AWS te manda un rack físico a tu datacenter con servicios AWS corriendo on-prem.

## Para el examen
- Hybrid cloud + quiero usar APIs de AWS on-prem → Outposts
- Latencia ultra baja a sistemas on-prem → Outposts
- Data residency (datos no pueden salir del datacenter) → Outposts
- S3 on Outposts: storage class especial, SSE-S3 por defecto

## Servicios disponibles
EC2, EBS, S3, EKS, ECS, RDS, EMR

## Diferencia clave
- El rack está en TU datacenter
- VOS sos responsable de la seguridad física
- AWS lo gestiona remotamente
