# 6.4 Amazon S3 - Storage Class Analysis

## Qué es
- Analiza uso de objetos y recomienda cuándo mover a otra storage class
- También llamado "S3 Analytics"

## Limitaciones
- Solo analiza: Standard ↔ Standard IA
- NO funciona para: One-Zone IA, Glacier

## Características
- Reporte diario
- Tarda 24-48h en empezar a mostrar datos
- Se puede visualizar en QuickSight

## Para qué sirve
- Primer paso antes de crear/mejorar Lifecycle Rules
- Te dice el "sweet spot" de cuándo mover objetos
