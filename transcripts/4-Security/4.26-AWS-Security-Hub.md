# 4.26 AWS Security Hub

**Curso:** AWS Solutions Architect Professional (SAP-C02)
**Sección:** 4 - Security
**Duración:** ~4 min

---

## Resumen

Security Hub = dashboard central que agrega findings de todos los servicios de seguridad (GuardDuty, Inspector, Macie, Config, etc). Multi-cuenta. Requiere Config habilitado. Genera eventos en EventBridge.

---

## Qué agrega Security Hub

```
GuardDuty ────┐
Inspector ────┤
Macie ────────┤
Config ───────┼──► SECURITY HUB ──► Dashboard
Firewall Mgr ─┤         │
IAM Analyzer ─┤         └──► EventBridge
Systems Mgr ──┤
Health ───────┘
```

---

## Flujo completo

```
Múltiples cuentas
       │
       ▼
┌─────────────────┐
│  SECURITY HUB   │──► Dashboard (findings centralizados)
│  (agrega todo)  │──► EventBridge (automatización)
└────────┬────────┘
         │
         ▼
  Amazon Detective (investigar origen)
```

---

## Puntos Clave

1. **Requiere AWS Config** habilitado primero
2. **Multi-cuenta** - agrega findings de todas
3. **Dashboard central** - un solo lugar para todo
4. **EventBridge** - para automatizar respuestas
5. **Detective** - para investigar el origen
