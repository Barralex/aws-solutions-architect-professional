# 4.7 RDS Security

**Curso:** AWS Solutions Architect Professional (SAP-C02)
**Sección:** 4 - Security
**Duración:** 1 min

---

## Resumen

Opciones de seguridad para RDS: encryption at rest (KMS), TDE para Oracle/SQL Server, SSL in-flight, IAM auth para MySQL/PostgreSQL/MariaDB.

---

## Opciones de Seguridad RDS

```
┌─────────────────────────────────────────────────────────────┐
│  ENCRYPTION AT REST                                         │
│                                                             │
│  • KMS encryption para EBS volumes y snapshots             │
│  • TDE (Transparent Data Encryption)                        │
│    └── SOLO Oracle y SQL Server                            │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  ENCRYPTION IN-FLIGHT                                       │
│                                                             │
│  • SSL encryption disponible para TODOS los engines        │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  IAM AUTHENTICATION                                         │
│                                                             │
│  • SOLO para: MySQL, PostgreSQL, MariaDB                   │
│  • NO para: Oracle, SQL Server                             │
│                                                             │
│  Nota: IAM es solo para AUTENTICACIÓN                      │
│        La AUTORIZACIÓN sigue siendo dentro de RDS          │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  OTROS                                                      │
│                                                             │
│  • Snapshot unencrypted → encrypted (se puede copiar)      │
│  • CloudTrail NO trackea queries dentro de RDS             │
└─────────────────────────────────────────────────────────────┘
```

---

## Tabla Resumen para el Examen

| Feature | Engines soportados |
|---------|-------------------|
| **KMS encryption** | Todos |
| **TDE** | Oracle, SQL Server |
| **SSL in-flight** | Todos |
| **IAM authentication** | MySQL, PostgreSQL, MariaDB |

---

## Puntos Clave para el Examen

1. **TDE** = solo Oracle y SQL Server (el examen te puede engañar)
2. **IAM auth** = solo MySQL, PostgreSQL, MariaDB (NO Oracle)
3. **CloudTrail** = NO sirve para ver queries de RDS
4. **Snapshot** = se puede copiar de unencrypted a encrypted
