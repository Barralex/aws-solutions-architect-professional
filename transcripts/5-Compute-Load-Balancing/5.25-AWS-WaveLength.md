# 5.25 AWS WaveLength

## TL;DR
AWS en el datacenter de una telco 5G. Ultra baja latencia para apps móviles.

## Para el examen
- 5G en la pregunta → WaveLength
- Ultra baja latencia para móviles → WaveLength
- AR/VR, gaming móvil, connected cars → WaveLength

## Qué es
- Infraestructura AWS dentro de redes 5G de telecos
- El tráfico nunca sale de la red del carrier
- Se conecta a parent region si necesitás RDS/DynamoDB

## Servicios disponibles
EC2, EBS, VPC

## Casos de uso
- Smart Cities
- Connected Vehicles
- AR/VR
- Real-time Gaming
- Live Video Streaming
