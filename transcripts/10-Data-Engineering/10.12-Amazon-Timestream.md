# 10.12 Amazon Timestream

## Qué es

- **Time series database** serverless
- Fully managed, auto-scaling
- Trillones de eventos/día
- SQL compatible
- Más rápido y barato que RDS para time series

## Storage tiers

- **Memoria**: datos recientes (rápido)
- **Storage optimizado**: datos históricos (barato)

## Integraciones

### Producers:
- AWS IoT
- Kinesis Data Streams → Lambda
- Kinesis Data Streams → Flink
- Amazon MSK → Flink
- Prometheus, Telegraf

### Consumers:
- Amazon QuickSight (dashboards)
- Amazon SageMaker (ML)
- Grafana
- Cualquier app JDBC/SQL

## Casos de uso

- IoT
- Operational monitoring
- Real-time analytics
- Métricas, sensores, logs con timestamp
