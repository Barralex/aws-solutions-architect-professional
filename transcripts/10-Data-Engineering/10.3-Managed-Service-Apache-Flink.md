# 10.3 Amazon Managed Service for Apache Flink

## Nombre anterior
- Kinesis Data Analytics for Apache Flink

## Qué es

- Framework para procesar streams en **real-time**
- Lenguajes: Java, SQL, Scala
- Cluster managed por AWS

## Sources (de dónde lee)

- Kinesis Data Streams ✅
- Amazon MSK (Kafka managed) ✅
- **Firehose ❌ NO**

## Features managed

- Provisioning automático de compute
- Parallel computing
- Auto-scaling
- Backups (checkpoints + snapshots)

## Para el examen

| Ves en pregunta | Respuesta |
|-----------------|-----------|
| Procesar streams real-time con SQL/Java | Managed Service for Apache Flink |
| Analytics sobre Kinesis Streams | Managed Service for Apache Flink |
| Flink lee de Firehose? | NO, solo Streams y MSK |
| Transformaciones complejas en streaming | Flink (más poder que Lambda) |
