# 6.10 AWS DataSync - Solution Architecture

## DataSync con Direct Connect (acceso privado)

### Problema
Tenés Direct Connect y querés que DataSync NO use internet público.

### Dos opciones

```
OPCIÓN 1: Public VIF (NO privado)
─────────────────────────────────
DataSync Agent → Direct Connect → Public VIF → Internet → DataSync URL pública
                                                              │
                                              (sale a internet, no queremos esto)


OPCIÓN 2: Private VIF + VPC Endpoint ✅ (privado)
────────────────────────────────────────────────
DataSync Agent → Direct Connect → Private VIF → VPC → Interface Endpoint → DataSync
                                                              │
                                              (todo privado, nunca sale a internet)
```

## Para el examen

```
DataSync + Direct Connect + privado → VPC Interface Endpoint (PrivateLink)
```
