# 4.20 Amazon Inspector

**Curso:** AWS Solutions Architect Professional (SAP-C02)
**Sección:** 4 - Security
**Duración:** ~3 min

---

## Resumen

Inspector = escaneo automático de vulnerabilidades para EC2, ECR (imágenes Docker) y Lambda. Busca en base de datos CVE, escanea continuamente, reporta a Security Hub y EventBridge.

---

## Qué escanea Inspector

| Recurso | Qué busca | Cómo |
|---------|-----------|------|
| **EC2** | Vulnerabilidades OS + accesibilidad de red | SSM Agent |
| **ECR** | Vulnerabilidades en imágenes Docker | Al hacer push |
| **Lambda** | Vulnerabilidades en código + dependencias | Al desplegar |

---

## Flujo

```
EC2 (SSM Agent)  ──┐
ECR (push image) ──┼──► INSPECTOR ──► Security Hub
Lambda (deploy)  ──┘         │
                             └──► EventBridge (automatización)
```

---

## Puntos Clave

1. **Solo 3 recursos:** EC2, ECR, Lambda
2. **Escaneo continuo** - se re-ejecuta si CVE database se actualiza
3. **Risk score** para priorizar vulnerabilidades
4. **Reporta a:** Security Hub + EventBridge
