# 10.11 DocumentDB

## Qué es

- **Aurora pero para MongoDB**
- NoSQL, JSON documents
- Fully managed, HA, 3 AZ replication
- Storage auto-grows en incrementos de 10GB
- Millones de requests/segundo

## Arquitectura (igual que Aurora)

- Primary instance (read/write)
- Replica instances (read)
- Shared storage layer (3 AZ)

## Pricing (todo provisioned, NO on-demand tier)

- On-demand instances (por segundo, mín 10 min)
- Database I/O (por millón de IOPs)
- Database storage (GB/mes)
- Backups en S3 (GB/mes)
