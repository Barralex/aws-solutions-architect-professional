# 4.27 Amazon Detective

**Curso:** AWS Solutions Architect Professional (SAP-C02)
**Sección:** 4 - Security
**Duración:** ~2 min

---

## Resumen

Detective = investigar causa raíz de problemas de seguridad. Usa ML y gráficos. Procesa VPC Flow Logs, CloudTrail, GuardDuty. Te da visualizaciones para entender qué pasó.

---

## El problema

GuardDuty dice "hay amenaza", Security Hub muestra el finding... pero ¿DE DÓNDE VINO? ¿QUÉ PASÓ EXACTAMENTE?

Investigar manualmente = revisar CloudTrail, VPC Flow Logs, correlacionar datos... horas o días.

---

## La solución: Detective

Usa ML para correlacionar automáticamente:
- VPC Flow Logs
- CloudTrail
- GuardDuty findings

Te da visualizaciones para llegar rápido a la causa raíz.

---

## Flujo

```
GuardDuty: "Hay amenaza!"
        │
        ▼
Security Hub: "Acá está el finding"
        │
        ▼
Detective: "Esto es lo que pasó, de dónde vino,
           quién lo hizo, cuándo empezó..."
```

---

## Puntos Clave

1. **Investigar causa raíz** - el "detective" del nombre
2. **ML + gráficos** - correlaciona datos automáticamente
3. **Inputs:** VPC Flow Logs, CloudTrail, GuardDuty
4. **Output:** Visualizaciones para entender el problema
