# 6.11 AWS Data Exchange

## Qué es
Marketplace para comprar/vender datos de terceros.

## Providers de ejemplo
| Provider | Data |
|----------|------|
| Reuters | 2.2M+ news stories/año |
| Change Healthcare | 14B healthcare transactions |
| Dun & Bradstreet | 330M business records |
| Foursquare | 220M customers, 60M venues |

## Cómo funciona

```
Proveedor (ej: Foursquare)
         │
         ▼
   Data Exchange ──API──► S3 (tu cuenta)
                              │
                              ▼
                    SageMaker / Athena / etc.
```

## Variantes

### Data Exchange for Redshift
```
Data Exchange → Redshift (directo, sin pasar por S3)

También al revés:
Tu data en Redshift → Licenciar vía Data Exchange
```

### Data Exchange for APIs
- Suscribirse a APIs de terceros
- Usa AWS SDK (autenticación y governance nativo)

## Para el examen

```
Comprar datasets de terceros     → Data Exchange
Vender tu data a otros          → Data Exchange
Third-party data directo a S3   → Data Exchange
```
