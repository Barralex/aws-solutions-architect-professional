# 5.26 AWS Local Zones

## TL;DR
Mini-AZs en ciudades que no tienen región AWS completa. Para baja latencia.

## Para el examen
- Usuarios en ciudad sin región AWS + baja latencia → Local Zones
- Extender VPC a otra ciudad → Local Zones

## Qué es
- Extensión de una región a ciudades cercanas
- Se habilitan manualmente (no vienen por defecto)
- Extendés tu VPC creando subnets en la Local Zone

## Ejemplo
```
us-east-1 (Virginia)
├── AZ-1a
├── AZ-1b
├── AZ-1c
└── Local Zones (opcionales):
    ├── Boston
    ├── Chicago
    ├── Miami
    └── etc.
```

## Servicios disponibles
EC2, RDS, ECS, EBS, ElastiCache, Direct Connect

## Diferencia con otros
- Outposts: AWS en TU datacenter
- WaveLength: AWS en red 5G de telco
- Local Zones: AWS en ciudades sin región completa
